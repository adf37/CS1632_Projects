How to Build
------------------------------------------------------
1. Clone the repo to your local desktop 
2. All of the code needed will be located in SlowLifeGUI_modified folder including the JUnit
test file called TestProfiler.java
3. All necessary jars should be a part of the eclipse project. Run the TestProfiler.java file
as a JUnit test.
4. All tests should pass testing the 3 different methods that were refractored in from the original 
files.
